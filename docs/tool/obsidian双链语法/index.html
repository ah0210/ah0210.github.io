# 在笔记A中链接到笔记B
[[笔记B]]

# 带显示文本的链接
[[笔记B|自定义显示文本]]

# 链接到特定标题
[[笔记B#具体标题]]

# 链接到特定段落
[[笔记B#^段落ID]]

搜索替换
	使用`Ctrl+Shift+F`全局搜索
	搜索特定关键词
	替换为`[[$1]]`格式
