飞雪连天过大年

辉哥 发布于 2000-01-25 收录于 生活

这是我第一次在外地过年，是在有花城之称的广州，第一次过没有雪的新年。
如果不是亲身生活在这个四季如春的城市，我很难想象腊月里会没有雪。
我的家乡在黑龙江，那是个四季分明的地方。
春节正是十冬腊月，也差不多是最冷的日子。
过了春节，春天就快到了。
借用金大侠的话——飞雪连天，可能并不是很确切，但是家乡的新年确实有自己独特的韵味。
相对南方，飞雪连天也许正是它的特征。
