人生老照片

辉哥 收录于 生活
2026-01-01 2026-01-11 约 600 字 预计阅读 2 分钟 - 次阅读   - 条评论

原来那是一生里最好的岁月……
那是八十年代的某天吧，当时家里的日子过得还好。
爸爸妈妈都那么年轻，孩子们在成长，最小的孩子，也上了小学，日子充满了奔头。
文革结束了，爸爸从下放的林场回到城里正式上班，当时全国拨乱反正，在城里有份工作，有固定的收入，大家的日子过得都不错，正是中国人在所有日子里，最幸福的一段时间。
孩子们每年基本都能做一套新衣服，虽然都是普通的粗布衣服，但在当时都是难得了。
再后来的几年，当改革开放以后，生活逐渐艰难起来，即使这些普通的粗布衣服也不能每年都新作了。
回想起来，文革结束，改革开放初期真的是最幸福的日子啊。
弟弟的大檐帽，我的军帽，妹妹的花衣服——那也是差不多我们穿的最好的日子。
爸爸妈妈那么年轻！
妈妈那么漂亮，爸爸那么帅气，那是我永远的记忆。
人生的际遇啊，就像大河里的一粒沙吧，浮沉翻滚，谁知道最后，会去向哪里哪。
当年爸爸妈妈精心装饰了一个相框，里面有很多的照片。
而这几张照片，是后来在房间的角落里，大部分照片已经被水浸湿了，好像当时已经没人再在乎了，我把他们精心的保存起来，直到今天从新翻出来。
总以为来日方长，其实，来日并不方长。
今天我深深理解，人生的幸福与不幸，全来自于认知，但是有些事情已经来不及了。
珍惜当下，不管是什么，照片或者任何其他的东西，当下就分享，当下就说，当下就做，不要等以后，不要等将来……

妈妈和妹妹在天国幸福快乐！
